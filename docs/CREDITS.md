## Third-Party Code

### Color Picker Implementation

The Corel color picker in CHROMATIC is based on the **Corel Color Picker** from the [Every Color Picker](https://github.com/pshihn/every-color-picker) project by **Preet Shihn**.

- **Author**: Preet Shihn
- **Project**: Every Color Picker
- **Repository**: https://github.com/pshihn/every-color-picker
- **License**: MIT License

I adapted and simplified the original TypeScript implementation to create a compact JavaScript version that fits CHROMATIC's circular design. The core concepts of the triangle gradient rendering and hue ring interaction are derived from Preet's excellent work.

### About Preet Shihn

Preet Shihn is a developer and creator known for innovative web-based tools and libraries. Check out more of his work:

- **GitHub**: https://github.com/pshihn
- **Every Color Picker Demo**: https://everycolorpicker.com

His work has been instrumental in making color selection more intuitive and visually appealing across the web.

# CHROMATIC

A floating circular desktop app for color work. No title bar, no window borders, just a circle that hovers on your screen.

## What is it?

CHROMATIC is a lightweight color tool built for designers and developers. It lives on your desktop as a floating circle, always accessible, never in the way. Built with Electron.

## Tabs

| # | Tab | Description |
|---|-----|-------------|
| 1 | Hex previewer | Preview colors and convert between colortypes |
| 2 | Color library | Save and organize colors and palettes |
| 3 | Pipet tool | Pick colors directly from your screen |
| 4 | Tint generator | Generate light and dark variants of a color |
| 5 | Kleur harmony | Explore complementary and harmonic color combinations |
| 6 | Gradient builder | Build gradients and export CSS code |

## Tech stack

- [Electron](https://www.electronjs.org/), desktop app framework
- Vanilla JavaScript, HTML, CSS
- Local JSON storage via Node.js `fs` module

## Getting started

```bash
git clone https://github.com/TxT14/CHROMATIC.git
cd CHROMATIC
npm install
npm start
```
